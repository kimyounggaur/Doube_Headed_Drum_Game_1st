# 조선의 북소리: 전장의 지휘자

> **리듬은 곧 전술이다 (Rhythm is Tactics)**

한국 전통 장구 장단을 활용한 **리듬 전략 액션 RPG** 웹 프로토타입입니다.
플레이어는 칼을 든 장수가 아닌, **군영의 신호를 담당하는 고수(鼓手)**가 되어
북소리로 부대를 지휘합니다.

---

## 게임의 영혼

- **장르**: 리듬 전략 액션 RPG
- **배경**: 임진왜란 · 병자호란 등 조선 중기 (판타지 요소 없음)
- **차별점**: "북소리가 군대를 움직였던 역사적 사실"에 기반한 묵직한 전쟁 드라마

### 불변의 게임 문법

| 구음 | 타법 | 명령 | 효과 |
|------|------|------|------|
| 궁(Gung) | 궁편 강 | 보병 전진/방패벽 | 기병 돌격 저지 |
| 구(Gu) | 궁편 약 | 보병 대기/재정비 | 체력 회복 |
| 따(Tta) | 채편 강 | 궁수 일제 사격 | 화망 구성 |
| 다(Da) | 채편 약 | 조준/장전 | 다음 사격 치명타↑ |
| 덩(Dung) | 양손 강 | 전군 총공격 | 최대 데미지 |
| 더(Deo) | 양손 약 | 전군 후퇴/산개 | 광역 공격 회피 |

4대 진법(학익진·일자진·장사진·원진)은 위 타법들의 연속 패턴으로 발동됩니다.

---

## 기술 스택

- **빌드**: Vite + React 18 + TypeScript (strict)
- **2D 렌더링**: PixiJS 8 + @pixi/react
- **오디오**: Tone.js (단일 AudioContext로 리듬 동기화 보장)
- **상태관리**: Zustand
- **스타일**: Tailwind CSS + CSS Modules (한국 전통 단청 팔레트)
- **애니메이션**: Framer Motion (UI), GSAP (PIXI 보조)
- **검증**: Zod
- **테스트**: Vitest
- **국제화**: react-i18next (한국어 우선, 영어 대비)

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 (기본 포트: 5173)
npm run dev

# 타입 체크
npm run typecheck

# 린트
npm run lint

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview

# 테스트
npm run test
```

브라우저에서 `http://localhost:5173`을 열면 메인 메뉴가 나타납니다.

## 디자인 원칙

1. **음악적 정확성**: BPM과 박자 구조를 코드 정확성만큼 엄수
2. **역사 고증**: 한복·갑주·진법 묘사는 사실 기반 (철릭·두정갑 등)
3. **단청 미감**: 적·청·황·녹·백 5색과 한지 질감 활용, 그라데이션 최소화
4. **모바일 우선**: 9:16 세로화면을 기본 레이아웃으로 설계
5. **접근성**: 청각 장애인을 위한 시각적 메트로놈 옵션 필수
